# Global Cost of Living Index Analysis 📊🌍

## 📌 Project Overview
This project provides a comprehensive Exploratory Data Analysis (EDA) of the global cost of living in 2024. Using a dataset of various economic indices, I analyzed how different factors like rent, groceries, and restaurant prices correlate with local purchasing power across different countries.

## 🚀 Key Insights & Features
* **Data Cleaning & Preprocessing:** Handled global economic datasets to ensure accuracy in multi-country comparisons.
* **Advanced Visualizations:** Created insightful charts using **Seaborn** and **Matplotlib** to show the distribution of living costs.
* **Purchasing Power Analysis:** Investigated the "Salary Strength vs. Groceries Cost" to identify which countries offer the best quality of life relative to income.
* **Correlation Mapping:** Used heatmaps to visualize the relationships between Rent Index, Cost of Living, and Local Purchasing Power.

## 🛠️ Tech Stack
* **Language:** Python.
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn.
* **Environment:** Jupyter Notebook (`.ipynb`).

## 📂 Dataset
The analysis is based on the **Cost of Living Index by Country 2024** dataset, which includes metrics like:
* Cost of Living Index
* Rent Index
* Groceries Index
* Local Purchasing Power Index
